# ☕ Whisk & Wonder Café Website

A modern, aesthetic café website designed to deliver a delightful digital experience inspired by cozy café vibes and elegant UI design.

This project focuses on smooth interactions, visually appealing layouts, and a clean user experience — perfect for showcasing frontend development skills.

---

## ✨ Features

• Beautiful **café-themed UI design**  
• Smooth scrolling and modern layout  
• Interactive sections with clean animations  
• Fully responsive design for all devices  
• Elegant typography and color palette  
• Menu showcase with structured layout  
• Call-to-action buttons for engagement  
• Optimized for performance and usability  

---

## 🛠 Tech Stack

- **HTML5**
- **CSS3**
- **TypeScript**
- **Next.js**
- **Responsive Design**
- **Modern UI/UX Principles**

---

## 📂 Sections

The website includes the following sections:

- **Hero** – Eye-catching landing section with branding and CTA  
- **About** – Introduction to the café and its story  
- **Menu** – Food & beverage offerings with structured layout  
- **Specials** – Highlighted items or featured products  
- **Gallery** – Visual showcase of café ambiance and dishes  
- **Testimonials** – Customer feedback and trust-building  
- **Contact** – Location, contact details, and inquiry section  

---

## 🎯 Purpose of the Project

This project was built to:

• Practice real-world website design  
• Improve frontend development skills  
• Create a visually appealing business website  
• Understand layout structuring and responsiveness
